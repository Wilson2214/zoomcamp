# Github setup
1. Create remote repo
2. In local repo run git init
3. git add . (add files to gitignore to prevent adding)
4. git commit -m "First commit"
5. git remote add origin <remote url>
6. git push origin main
7. git branch --set-upstream-to=origin/main main
